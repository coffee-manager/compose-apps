앱오더 자동수락 프로그램

힘들게 포스기까지 뛰어갈 필요 없어요!

자동으로 모든 앱오더 주문을 수락해줘요.

32bit 다운로드 64bit 다운로드

(포스기에 다운로드 및 설치)

설치 방법

  1. 프로그램 다운로드

    대부분 32bit 버전 다운로드
    신규매장 64bit 버전 다운로드

    Tip: 내컴퓨터(내PC)➝ 우클릭➝ 속성➝ 시스템 종류➝ 32/64 확인

    Tip: 두 버전 모두 다운 후 시도해보셔도 됩니다.

  2. 압축 해제

    - 파일에 우클릭 ➝ 압축 풀기 선택

    - 파일을 길게 터치하면 우클릭이 됩니다.

    - 혹은 파일 더블 클릭 후 압축풀기 클릭

    ▼(아래 이미지 참조)

  3. setup 파일 더블클릭

    ▲ 위 이미지 처럼 나온다면 정상 설치 완료

  4. 바탕화면에 '앱오더 자동수락' 파일 실행

    위 파일 더블클릭

    위 이미지처럼 실행되면 확인 완료

문제 해결

1. 프로그램은 정상적으로 실행되었으나
앱오더가 자동수락 안되는 경우


[확인 방법]

1. 앱오더 자동수락 실행파일에 우클릭
2. '관리자 권한' 으로 실행
3. 정상적으로 동작하는지 확인
정상적으로 동작하면 '권한' 문제 입니다.


[해결 방법]

1. 앱오더 자동수락을 항상 '관리자 권한' 으로 실행
2. 포스 프로그램을 일반 권한(관리자권한x) 으로 실행
위 두가지 중 하나의 방법으로 해결됩니다.

2번의 경우, 셀프 설정이 어려우시다면
포스뱅크에 문의하여 조치 받을 수 있습니다.

2. 그외

상단의 '개발자 문의' 통해서 말씀 주세요.

어디까지 진행하셨는지,
어떻게 안되시는지 구체적으로 말씀해주실수록
더욱 빠른 조치가 가능합니다.
