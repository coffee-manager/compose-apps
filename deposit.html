Made By 컴포즈커피 강북번동점 점주님

설치 방법

  1. 프로그램 다운로드

    대부분 32bit 버전 다운로드
    신규매장 64bit 버전 다운로드

    Tip: 내컴퓨터(내PC)➝ 우클릭➝ 속성➝ 시스템 종류➝ 32/64 확인

    Tip: 두 버전 모두 다운 후 시도해보셔도 됩니다.

  2. 압축 해제

    - 파일에 우클릭 ➝ 압축 풀기 선택

    - 파일을 길게 터치하면 우클릭이 됩니다.

    - 혹은 파일 더블 클릭 후 압축풀기 클릭

    ▼(아래 이미지 참조)

  3. setup 파일 더블클릭

    ▲ 위 이미지 처럼 나온다면 정상 설치 완료

사용 방법

간략한 특징 및 기능 소개

1. 프린터 설정(필수)

프린터가 정상적으로 설정되면
메뉴 선택 후 결제시 프린터에서 주문서가 출력됩니다.
(포스기 결제와 비슷)

그외
1) 이전 거래내역 출력
2) 영수증 출력
3) 주문서 재출력

2. 음료 검색 기능

메뉴 찾는 시간 낭비는 이제 그만!
아주 쉽고 빠른 검색 기능 제공

축약어, 카테고리, 자음, 키워드 검색 등을 지원
한/영 실수 등을 보조

- 간편 검색

- 한/영 실수 보조

- 자음 검색


3. 입금/출금

1) 입금 실수를 줄이기 위한 한글 표기
2) 현금 / 카드 결제에 따른 추가금 자동 계산(옵션에서 변경 가능)
3) 음수 입력시 결제 금액 차감 가능
